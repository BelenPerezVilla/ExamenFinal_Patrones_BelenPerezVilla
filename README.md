# Proyecto final del simulador de cocina dinamico
# *ExamenFinal_Patrones_BelenPerezVilla*
**Nombre y numero de cotrol: Belen Perez Villa, 21212579**
El proyecto consiste en un simulador de cocina, donde el chef puede registrar recetas base e ingredientes adicionales, y el cliente puede seleccionar una receta y personalizarla agregando ingredientes extra según su preferencia. El sistema calcula automáticamente el costo total y genera un resumen del pedido final.
